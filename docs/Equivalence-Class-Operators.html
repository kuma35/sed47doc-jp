15.3.6.2等価クラス演算子 ([= … =])

正規表現はリスト内の等価クラス式を認識します。同値クラス式は、すべて同じ同値クラスに属する一連の照合要素です。オープン等価クラス演算子とクローズ等価クラス演算子の間に照合要素を入れることで、等価クラス式を作成します。‘[=’はオープン等価クラス演算子を表し、‘=]’はクローズ等価クラス演算子を表します。例えば、もし‘a’と‘A’が等価なクラスならば、‘[[=a=]]’と‘[[=A=]]’の両方は、‘a’と‘A’の両方にマッチします。同値クラス式の照合要素が同値クラスの一部ではない場合、正規表現マッチャーは同値クラス式を照合記号と見なします。
